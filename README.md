# VerSort

![License](https://img.shields.io/github/license/Tecnativa/versort)

Sort versions according to different versioning schemas.

## Install

To use as a CLI app:

```sh
pipx install versort
```

To use as a library:

```sh
pip install versort
```

## Usage

### CLI

```sh
echo 2 2a1 v1 | versort pep440
```
